DESCRIPTION:

OneSport is dedicated to offering up-to-date and pertinent information on a wide range of sports and athletic events. 
Our primary aim is to keep fans and enthusiasts well-informed and engaged. 
The project strives to serve as a one-stop destination for comprehensive sports news, event updates, and a convenient platform for purchasing tickets to various sporting events.

USER STORIES:

As a User, I want to be able to signup in order to have an account 
As a User, I want to be able to login
As a User, I want to logout from the account by clicking on the Navbar
As a User, I want to be able to have access to my profile settings
As a User, I want to be able to delete my account by clicking on a button from the profile settings page
As a User, I want to be able to modify my email and password in order to change my credentials
As a User, I want to see all recent news
As a User, I want to filter Italian news by selecting from the menu of available countries 
As a User, I want to add to my favourite news all the news I'm interested in, in order to read them later
As a User, I want to delete from my favourite news all the news that I'm no longer interested 
As a User, I want to see available tickets for upcoming sporting events
As a User, I want to filter Italian tickets by selecting from the menu of available countries 
As a User, I want to add to my collection all the tickets I'm interested in, in order to purchase them later
As a User, I want to delete from my collection all the tickets that I'm no longer interested 
As a User, I want to be able to delete all the news and tickets in my favourite list
As a User, I want to send a friend request to another user, in order to add him to my friend list
As a User, I want to check if someone has sent me a friend request, in order to accept or delete it
As a User, I want to delete a friend from my friend list

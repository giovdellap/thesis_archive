Endpoints:

Endpoint URL | Request Object | Response Object | HTTP Method
-------------------------------------------------------------
/ | None | UsersList | GET
/get_by_email/<string:email> | None | User | GET
/<int:user_id> | None | User | GET
/login | LoginInput | LoginResponse | POST
/checkToken | TokenInput | TokenResponse | GET
/signup | SignupInput | SignupResponse | POST
/logout | None | LogoutResponse | POST
/<int:user_id>/edit | UpdateUserInput | UpdateUserResponse | PUT
/<int:user_id> | None | DeleteUserResponse | DELETE
